AI Agent Prompt Mantığı

  • available_days = deadline − bugün (takvim günü)
  • target_tasks = round(available_days × 0.6) → min 3, max 45
  • Her görev = 1 odak seansı (focusPeriod dakika)
  • Görev isimleri spesifik: "HTML skeleton yaz: header, nav, hero" (❌ "HTML yap" değil)
  • Bilişsel yük: high (yaratıcı/derin) / medium (orta) / low (mekanik)

Bu Oturumda Düzeltilen Kritik Bug'lar

  • 🔴
    SyntaxError (JavaScript tamamen çalışmıyordu): saveProfile() fonksiyonu async olmadan await kullanıyordu. Tüm script parse edilemiyordu, hiçbir buton çalışmıyordu. → async function saveProfile() olarak düzeltildi.
  • 🔴
    Takvim etkinlikleri yüklenmiyor: FullCalendar Google Calendar plugin sadece API key ile private takvimi okuyamıyor. → gapi.client.calendar.events.list() ile custom events function yazıldı.
  • 🔴
    OAuth redirect döngüsü: PKCE code exchange tamamlanmadan getSession() null dönüyor, sürekli auth.html'e yönlendiriyordu. → INITIAL_SESSION event ile çözüldü.
  • 🔴
    Google Client Secret eksik: Supabase Google provider'da Client Secret girilmemişti, token exchange başarısız oluyordu. → Google Cloud Console'dan yeni secret oluşturulup Supabase'e eklendi.
